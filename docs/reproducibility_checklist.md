# Reproducibility Checklist

## Input Files Required
- Dataset files stored in the `data/` folder

## Scripts to be Executed
- Python scripts located in the `src/` folder

## Execution Order
1. Load dataset  
2. Perform data cleaning  
3. Generate visualizations  
4. Save outputs in `results/` folder  

## Expected Output Files
- Cleaned datasets  
- Graphs and charts  
- Analysis results stored in `results/`

## Software Dependencies
- Python  
- pandas  
- numpy  
- matplotlib  

## Assumptions
- Data is available in correct format  
- Required libraries are installed  
- Scripts are executed in the correct order  

## Limitations
- Does not include automated pipeline  
- Manual execution required  
- Limited scalability for large datasets
