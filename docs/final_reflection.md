# Final Reflection

## 1. How does Git help in maintaining research traceability?
Git helps track every change made in the project through commits. Each commit records what was changed, when, and by whom, making it easy to trace the evolution of research work and revert to previous versions if needed.

## 2. Why is GitHub useful for academic collaboration?
GitHub provides a platform for sharing code, data, and documentation. It enables multiple researchers to collaborate, review changes, and contribute efficiently using features like branches and pull requests.

## 3. Why should researchers create branches instead of working directly on main?
Branches allow researchers to work on new features or experiments without affecting the main stable version. This reduces the risk of breaking the main project and keeps the workflow organized.

## 4. What is the role of Pull Requests in collaborative work?
Pull Requests allow changes made in a branch to be reviewed before merging into the main branch. They help maintain quality, enable discussion, and ensure that all updates are properly validated.

## 5. Why are reproducibility files important in research repositories?
Reproducibility files provide clear instructions on how to replicate the results. This is essential in research to verify findings and ensure transparency.

## 6. What is the purpose of creating a version tag such as v1.0.0?
A version tag marks a stable and significant state of the project. It helps in identifying specific versions used in experiments, publications, or reports.

## 7. Why should a repository include a citation file?
A citation file allows others to properly reference the work in academic publications. It ensures that the author receives credit for their contribution.

## 8. How does this semester’s Git/GitHub workflow support better PhD research practice?
This workflow promotes structured development, proper documentation, version control, and reproducibility. It helps in maintaining organized research, improving collaboration, and ensuring that results can be validated and reused.
