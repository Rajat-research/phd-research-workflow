# Methodology

## Purpose of the Dataset
The dataset is used to analyze and understand patterns in the research data. It helps in deriving meaningful insights and supporting the research objectives.

## Handling Missing Values
Missing values were handled using techniques such as removal of null entries or imputation using mean/median values.

## Reason for Choosing This Method
These methods were chosen because they are simple, effective, and suitable for structured datasets without introducing significant bias.

## Visualizations Generated
- Bar charts  
- Line graphs  
- Histograms  

These visualizations help in understanding trends, distributions, and relationships in the data.

## Limitations
- Assumes data is clean and structured  
- Basic imputation may not capture complex patterns  
- Limited use of advanced statistical or machine learning techniques
