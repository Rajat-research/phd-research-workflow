# Paper 2

## Citation
Van Noorden, R. (2014). Online collaboration: Scientists and the social network. Nature, 512(7513), 126–129.

## Summary
The paper discusses how online tools and platforms like GitHub help researchers collaborate globally.

## Key Points
- GitHub promotes open science
- Enables version control and collaboration
- Facilitates sharing of research outputs

## Relevance
This supports the use of GitHub in managing and sharing PhD research work.
