# PhD Research Workflow

## Project Title
A Structured Workflow for Managing PhD Research Using Git and GitHub

## Objective
The objective of this project is to organize and manage PhD research work efficiently using Git and GitHub for version control, collaboration, and documentation.

## Folder Structure

- docs/ → Contains documentation, reports, and notes
- src/ → Source code and scripts used in research
- data/ → Raw and processed datasets
- results/ → Outputs such as graphs, tables, and analysis results
- references/ → Literature notes and research paper summaries

## Tools Used
- Git
- GitHub
- VS Code / PyCharm

## Outcome
This workflow ensures proper version control, better organization, and easy tracking of research progress.
